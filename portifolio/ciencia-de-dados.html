Introdução

A Ciência de Dados extrai conhecimento de dados brutos para auxiliar na tomada de decisões. Através de ferramentas como o Google Colab e bibliotecas como Pandas, NumPy e Matplotlib, podemos analisar dados, identificar padrões e tomar decisões mais inteligentes. Conceitos como amostragem, tendências centrais e medidas de dispersão são essenciais para essa análise. A Ciência de Dados tem diversas aplicações, como a análise de mercado.

  • Google Colab: Plataforma utilizada para desenvolver e executar código Python em um ambiente de notebook.
  • Consulta e Coleta de Dados: Importação e análise de dados, exemplificada com a tabela de dados do Titanic.
  • Bibliotecas: Pandas, NumPy e Matplotlib
  • Amostragem: Técnicas de seleção de subconjuntos de dados para análise.
